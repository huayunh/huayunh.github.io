Alerting inside Bloomberg Terminal

Alerting inside Bloomberg Terminal

class

July 2019

Task

Design the interactions involved when the user is creating alerts for securities inside the Bloomberg Terminal. The user should be able to:

  • compare a given field to a constant value that the trader picks
  • compares a given field to another field

What I did

  • Understood the problem
  • Conducted secondary researches
  • Brainstormed for potential directions
  • Prototyped the wireframes
  • Provides directions for potential future iterations

Challenges

  • There are no user interviews: I have to rely on my speculations and what's available on the internet

What I learned

  • The data’s volatile nature is inducing the fluctuation problem.
  • The alert creation functionality is embedded into the Bloomberg Professional Service, making it possible to use other Bloomberg resources to gain better insights about the market.
