personalweb

Apirl 2017

A game I built using socket.io while learning Node.js. A crowd-sourced version of minesweeper, where the revealing of a grid is based on majority vote.

A minesweeper where you need at least half of the participants to "agree" on a cell before it is revealed. I was mostly curious about whether crowd sourcing would speed minesweeper up or not when I started working on this project, however later on my focus started to shift to how to make the interface as nice-looking as possible.
